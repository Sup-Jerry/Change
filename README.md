# 《在不可逆的世界里经营可逆空间》

这是一本关于个体在高不确定性时代中，如何保留选择权、重建行动能力的写作。  
全书聚焦于一个核心命题：当外部条件不可逆时，人仍可以通过认知、结构与节奏设计，经营出可逆空间。
